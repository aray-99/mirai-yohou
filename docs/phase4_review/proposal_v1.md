## 1. 現在地と Phase 4 の問い

Phase 3 完了時点の資産: 5 カ国(JPN/THA/KOR/TUR/EGY)全てで walk-forward
合格(#0069/#0082/#0091)、うち 3 カ国は是正 0 回の一発合格。国コードを渡せば
データ取得→較正→検証→予報 JSON→ダッシュボードが流れるパイプラインが実証済み。

現行モデルの構造的限界は 3 つで、Phase 4 の 3 課題に 1 対 1 対応する:

1. **世界が存在しない**: 各国は独立に走り、他国はいかなる経路でも影響しない。
   アラブの春(EGY 較正窓 2011〜21 の冒頭!)のような伝播現象は原理的に
   表現できない → #28。
2. **外生入力が定数**: wbar/netgrowth の較正窓定数フィット(§8.4 流儀)は
   1 年先検証を通るが、30 年外挿で w が観測トレンドと逆行する(#30 で診断済み。
   THA 観測 71.3→69.6% 低下中に予報が上昇へ転じる)→ #30。
3. **観測が遅い**: 年次統計 + 週次 ACLED(12 ヶ月エンバーゴ)のみで、
   数週間スケールの地政学的変化を吸収する観測チャネルがない → #29。

## 2. M14: 外生入力の時変化(UN WPP)— Issue #30

### 2.1 目的

wbar(t)・netgrowth(t) = b−d+mig を較正窓定数から UN World Population
Prospects(WPP)の国別系列に置換し、外挿領域の人口動態を「モデルの最も
予測可能な成分は外部の最良予測に従う」状態にする。

### 2.2 定式化

SPEC §5 は外生入力を最初から時間の既知関数として定義しており(式(1)(2))、
**支配方程式の変更はゼロ**。変更は experiments/M8_hindcast.jl の
fit_exogenous の置換のみ:

- netgrowth(t) = [b(t) − d(t) + mig(t)]、WPP 年次系列の区分線形補間。
- wbar(t) = WPP の生産年齢人口比率(15–64)射影。logit 変換してから区分線形
  補間(式(2)は logit 座標の回帰なので座標整合)。
- 追従率 kappa_w は現行どおり L2 較正対象のまま。wbar が時変になることで
  kappa_w の役割は「定数目標への回帰速度」から「射影トレンドへの追従遅れ」に
  変わるが、パラメータ数は増えない。

### 2.3 walk-forward 再現性(ヴィンテージ問題)への態度

厳密には、オリジン年 t0 の予報者は t0 時点で公表済みの WPP リビジョンしか
知り得ない。WPP は約 2 年毎改訂で過去版も入手可能だが、全オリジン×全国の
ヴィンテージ厳密化はコストが合格情報量に見合わない。コミットメント:

- **WPP 2024 単一リビジョンを凍結**して全用途(較正窓・予報)に使う。
  推計値部分(過去)は事実上観測と同一であり、射影部分の「未来のリビジョンを
  知っている」リークは人口動態の準決定論性ゆえ軽微 — と主張する。
- ただし主張を無検査で通さない: **THA 1 国でヴィンテージ感度検査**を実施
  (オリジン 2016〜18 の予報を WPP 2015 リビジョンで再実行し、被覆・RMSE の
  差分を DECISIONS に記録)。差が判定を覆す規模なら全体設計を再考する。
- サイドカーに WPP リビジョン番号・取得日を必ず記録(SPEC §0.5.6 拡張)。

### 2.4 合格基準の素案(検証ラン前に凍結)

1. 5 カ国 walk-forward 再実行で、プール被覆が全国 [0.80, 0.98] 帯内を維持。
2. w 系列の 1 年先 RMSE が定数近似版と同等以下(悪化許容 5% 以内)。
3. 10 年外挿での w トレンド方向(10 年差分の符号)が WPP 射影と 5/5 カ国一致
   (#30 の症状の直接解消確認)。
4. テスト green。是正上限 2 回(#0052 流儀)。基準・しきい値は不変更。

### 2.5 依存関係・工数

依存なし(単独で着手可能)。Issue 3〜4 本(WPP 取得スクリプト + 来歴、
fit_exogenous 置換、検証ラン、感度検査)。機械時間は 5 カ国再検証で
約 8〜10 時間。Fable 1 セッション + 委譲。

### 2.6 リスク

- kappa_w の較正値が定数 wbar 前提で凍結済み(#0063 等)。再較正するか
  凍結値を流用するかの判断が必要。**提案: 再較正はせず凍結値を流用**し、
  基準 1〜2 で性能劣化がないことを確認する(パラメータを動かす自由度を
  増やさない)。劣化したら是正 1 回目として国別再較正を発動。
- WPP に依存しない国は存在しないため可用性リスクは実質ゼロ。

## 3. M15: Hawkes 相互励起 — 国家間伝播(Issue #28)

Phase 4 の科学的本命。ただし「結合が推定できました」ではなく
「結合の有無を凍結された手続きで判定できる装置を作った」を成果と定義する。

### 3.1 定式化

国 i の興奮度 λ_e^i の力学(SPEC 式(13) + §6.3)を次のように拡張する:

```
d lam_e^i = -beta * lam_e^i dt
            + alpha0^i * (v^i/v0)^kappa_alphav dN^i_t          # 自己励起(現行)
            + gamma * sum_{j != i} W_ij * dN^j_t               # 相互励起(新規)
```

- **W = (W_ij) は事前凍結する近接行列**(行正規化、Σ_j W_ij = 1)。データから
  推定しない。構成は地理的隣接 + 地域ブロック共有(JPN–KOR、TUR–EGY を主結合、
  THA は現 5 カ国内で弱結合)を一次案とし、凍結前に出典(CEPII 距離データ等)を
  明記して DECISIONS に記録する。
- **推定する新規パラメータは γ ≥ 0 のスカラー 1 個のみ**。国ペア別
  α_{ji}(5 カ国で 20 自由度)は現データ量では識別不能と判断し、最初から
  置かない。
- 強度式 λ^i = min(λ̄, λ_b(X^i) + λ_e^i) は不変。Markov 埋め込みも保たれる
  (λ_e^i の状態は 1 国 1 次元のまま。外国イベントは入力として作用)。
- **安定条件の拡張**: 分岐行列 B、B_ii = α0^i/β、B_ij = γ W_ij / β(i≠j)の
  スペクトル半径 ρ(B) < 1 をパラメータ構築時に assert(現行の n = α0/β < 1 の
  自然な一般化)。余裕をもって ρ(B) ≤ 0.8 を推奨値とする。

### 3.2 推定と識別可能性

β を ACLED 系列単独で最尤推定した前例(SPEC §8.1)に倣い、
**γ もモデル本体から切り離して推定する**:

- データ: 5 カ国の週次 ACLED カウントパネル(既存の exclude_admin1 適用後、
  各国の較正窓の共通部分)。
- 尤度: 多変量 Hawkes のカウント尤度。過分散は各国の既存 ν プロファイル値で
  テンパリング(#0033 と同じ流儀を多国に横断適用)。
- **判定を事前凍結**: γ=0(独立)対 γ>0 の尤度比検定、有意水準 5%。加えて
  W の行を国間でランダム置換した並べ替え検定(200 回)で「γ>0 が W の構造を
  本当に使っているか」を確認。両方通って初めて結合を採用。
- **どちらの結果も成果**: 棄却できなければ「この 5 カ国・この期間では国家間
  伝播は検出されない」を DECISIONS に記録し、予報系は γ=0(現行と同一)の
  まま Phase 4 を閉じる。これは失敗ではなく、負の結果の凍結された確定である。

先回りの弱点開示: 現 5 カ国は伝播検出に不利な標本である。JPN/KOR は
低カウント、THA は域内で孤立、実質的な検定力は TUR–EGY ペア(アラブの春
2011 が EGY 較正窓に入る)にほぼ集中する。そこで:

- **手順ゲート 1(双子実験)**: 実データの前に、既知の γ で結合 5 カ国系を
  合成生成し、上記推定器が γ を復元でき・γ=0 データで偽検出しないことを
  E1 流儀(SPEC §13)で確認する。検定力が実データ相当のカウント水準で
  50% を下回るなら、実データ検定に進む前に「検出可能条件(国数・期間)」を
  報告してユーザー判断を仰ぐ。
- 国数追加(MENA 近隣等)が検定力を大きく改善するなら、M15 の中で
  データのみ追加(walk-forward 合格までは求めない「観測専用国」区分を新設)
  する選択肢を残す。観測専用国は較正 6 パラメータの推定を行わず、カウント
  パネルにのみ寄与する。

### 3.3 同化・予報・検証への接続(検証単位の変更)

結合系でも **EnKF は国別のまま**(ブロック対角)にする。これが M15 最大の
設計判断である:

- **同化(ヒンドキャスト)**: 外国イベントは観測データ(ACLED 時刻)として
  既知なので、λ_e^i への相互励起項は**決定論的な既知入力**になる。つまり
  同化は現行どおり 1 国ずつ実行でき、13n 次元の結合フィルタは不要。
  ExogenousEvents モード(§6.4)に「外国イベント時刻列」を追加するだけ。
- **予報(将来)**: ここで初めて多国同時シミュレーションが要る。5 カ国の
  アンサンブルメンバーを同一乱数世界で並走させ、内生発火が国境を越えて
  λ_e に加算される。VariableRateJump のスレッド並列不可(#0021)は結合系では
  そもそも逐次実行が意味的に必須なので制約が実害化しない(メンバー間並列は可)。
- **検証**: walk-forward の判定単位は従来どおり**国別**を維持する(被覆・RMSE
  基準は 1 国の観測系列に対する性質であり、結合の有無で定義が変わらない)。
  結合固有の追加基準を 1 つだけ立てる:
  **「結合予報の 5 カ国合算カウント対数尤度 ≥ 独立予報(γ=0)の合算値」**。
  独立モデルは常に走らせられるので、これは同一データ・同一シードでの
  A/B 比較として凍結できる。

### 3.4 合格基準の素案(凍結対象)

1. 双子実験ゲート: γ 復元(相対誤差 50% 以内)+ γ=0 での偽検出率 ≤ 5%。
2. 実データ判定: LRT + 並べ替え検定の事前凍結手続きの完遂(結果は問わない)。
3. γ>0 採用の場合のみ: 5 カ国 walk-forward 国別基準を全国維持 + 合算カウント
   尤度が独立版以上。1 国でも帯を割れば γ=0 へ差し戻し(差し戻しは是正に
   数えない — 採否の枝であって基準変更ではない)。
4. テスト green。是正上限 2 回。

### 3.5 依存関係・工数

M14 完了後に着手(30 年結合シミュレーションの土台整備)。Issue 6〜8 本
(多変量 Hawkes 尤度 + 検定、双子実験、ExogenousEvents 拡張、結合予報器、
検証ラン)。Fable 2〜3 セッション、機械時間 10〜15 時間(双子 + 5 カ国再検証)。

## 4. M16: 高頻度観測 — 金融市場系列(Issue #29 段階1)

### 4.1 目的と範囲

CDS スプレッド等の日次金融系列を σ_s(および間接的に λ)への観測として追加し、
エンバーゴなしの「速いセンサー」を得る。**#29 の 3 段階のうち段階 1 のみ**を
Phase 4 に入れる(段階 2 GDELT・段階 3 LLM 指標は §5 で除外)。

### 4.2 定式化(観測モデルへの組み込み)

- 一次系列: ソブリン CDS 5 年スプレッド(不在国は EMBI/国債利回りスプレッドで
  代替、選択規則は凍結時に国別確定)。
- 前処理: log 変換 → グローバル因子除去(全対象国横断の第 1 主成分、または
  米 HY スプレッドへの回帰残差 — 凍結時に一方を確定)→ §9.2 の規則どおり
  **週次平均へバッチ化**し、系列自己相関見合いで R を膨張。
- 観測作用素: `z_fin = a_i + b_i * sigma_s + eps`, eps 〜 N(0, R_fin)。
  σ_s の自然座標に対する線形写像(ξ_sig 座標では exp を介す弱い非線形 —
  log y と同程度で EnKF の暗黙線形化の範囲内)。
- **ν テンパリングとの整合**: テンパリング(#0033)はポアソン粒子重み側の
  機構であり、金融観測はガウスの EnKF 解析ステップ側に入るため機構的に直交。
  ただし両者が同じ潜在量 σ_s を引っ張るため二重計上のリスクはある。対策として
  金融観測は σ_s のみに作用させ(λ_e には作用させない)、カウント尤度側の
  定義は一切変更しない。

### 4.3 循環性の正直な処理(二段推定)

a_i, b_i は「σ_s の推定値に対する回帰」で決めるしかなく、その σ_s は
フィルタの産物 — 循環がある。隠さず手続きで断ち切る:

1. 第 1 パス: 金融観測**なし**の現行フィルタで較正窓の σ_s 経路を推定。
2. 較正窓内で z_fin を σ_s 推定値に回帰し、a_i・b_i・R_fin を**凍結**。
3. 第 2 パス以降(検証窓・予報)は凍結済み写像で金融観測を同化。
   検証窓で a_i, b_i を再フィットしない。

### 4.4 合格基準の素案(チャネル採否ゲート)

1. 変動 3 カ国(THA/TUR/EGY)の walk-forward で、金融観測ありの 1 年先
   カウント対数尤度が**なし版を 3 カ国とも厳密改善**。
2. 全 5 カ国で被覆帯 [0.80, 0.98] を維持(金融観測が区間を潰す自信過剰の検出)。
3. 満たさなければ**チャネル不採用を確定**し、診断を DECISIONS に記録して閉じる。
   M15 同様、負の結果も正式な完了である。

### 4.5 弱点の先回り開示

- σ_s は正規化(σ_Y=1)でスケールが規約的に決まる潜在量であり、b_i は
  国ごとに符号すら保証されない。回帰の決定係数が低い国では観測を追加しても
  R_fin が大きく実効的に無視される — それは正しい挙動であり、失敗ではない。
- CDS は流動性・グローバル risk-on/off の混入が大きい。グローバル因子除去を
  してなお残る共通変動は、M15 の相互励起と交絡し得る(市場は伝播を織り込む)。
  M16 を M15 の**後**に置くのはこのため(結合の有無を市場チャネルなしで
  先に確定しておく)。
- データ来歴: CDS は再配布制約のあるベンダーが多い。**再配布可能な出典
  (FRED・各国国債利回り等)で構成できない国はチャネル対象外**とする。
  来歴コミット主義(SPEC §0.5.6)を破ってまで導入しない。

### 4.6 依存関係・工数

M15 の判定確定後に着手。Issue 4〜5 本。機械時間 6〜10 時間。

## 5. やらないこと(Phase 4 スコープ外)

| 課題 | 理由 |
|---|---|
| フル構造結合(13n 次元 SDE + 結合ドリフト) | 年 1 点観測 × 結合パラメータ O(n²) で識別不能が確実(#0051/#0061 の 1 国内同定性ですら苦闘した実績)。#28 段階 3 のまま凍結保留 |
| LLM 由来指標(#29 段階3) | 観測生成器の再現性(同一入力→同一出力)と来歴主義が現行 LLM で担保できない。バックテスト用の過去系列を遡及生成すると「未来を知る要約者」のリーク検査が困難。金融チャネル(M16)の採否結果を見てから再訪 |
| #34 のモデル構造化(p の cleavage/ネットワーク拡張) | 13 変数の器そのものの変更であり、5 カ国合格資産の検証をすべて無効化する。Phase 5 以降の大改編候補として Issue 維持 |
| #34 の観測チャネル化(Polis/bridging スコア) | 概念的には #29 の枠に入るが、現 5 カ国で国政レベルの継続系列が存在しない(台湾 Polis 等は対象国外)。**データが存在しないものは設計しない**。対象国に系列が出現した時点で M16 の枠組みに追加候補として載る — 追加の設計は不要 |
| GDELT(#29 段階2) | ACLED 依存性の感度分析としての価値は認めるが、Phase 4 の 3 本柱いずれの合否にも寄与しない。バックログ維持(#0070) |

## 6. 検証の流儀との接続(総括)

Phase 2〜3 で確立した流儀 — 基準の事前凍結・是正上限 2 回・walk-forward・
負の結果の記録(#0030/#0052)— は全マイルストーンで不変。Phase 4 固有の
拡張は 2 点だけ:

1. **A/B 凍結比較の導入**(M15 §3.3、M16 §4.4): 新機構の採否は「同一データ・
   同一シードで新旧を並走させ、事前凍結した優劣基準で判定」する。基準帯への
   絶対判定(従来)に、対照付き相対判定が加わる。差し戻し(γ=0、チャネル
   不採用)は是正回数に数えない — 採否の枝は最初から手続きに含まれるため。
2. **検証単位**: 多国結合でも国別基準は国別のまま維持し、結合固有の性質
   (合算カウント尤度)だけを系全体の基準として追加する(§3.3)。「多国系に
   なったから既存基準を作り直す」ことはしない。
