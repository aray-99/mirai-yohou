# 実データフェーズ(Phase 2)設計提案 v0.1

第一段階(双子実験 E1、vM5/vM6)完了を受けた、実データ投入フェーズの設計たたき台。
**本文書は提案であり、⚖ 印の判断ポイントはユーザーの決定を要する。**
決定後、確定事項を SPEC への追補(または SPEC2)として固定し、E1 と同様に
検証基準を実行前に凍結する(DECISIONS 参照方式も踏襲)。

関連: SPEC §14(既知の限界)、DECISIONS #0016/#0018(k・tauA の持ち越し)、#0022。

---

## 1. ゴールと検証原則

- ゴール: 実データによるヒンドキャスト(過去再現)+ walk-forward 検証で、
  「予報円」出力の較正(被覆率)と情報量(自由ラン比)を実証する。
- §14-7「歴史は1回しか起きない」に従い、**単一国の単一期間で「検証済み」を
  主張しない**。walk-forward(訓練期間→検証期間を時間方向にずらして反復)
  × 複数国で擬似標本を稼ぐ。
- E1 の教訓を引き継ぐ: 判定基準・対象期間・国は**実行前に凍結**し、
  結果を見た後の変更は DECISIONS 必須(§0.5.8 と同じ手続き)。

## 2. 観測系列とデータソースの対応(§9.1 → 実データ)

| 観測 | 候補ソース | 頻度(実際) | 備考 |
|------|-----------|------------|------|
| log P | UN WPP / World Bank | 年次 | 安定。改定小 |
| logit w | UN WPP(15–64歳比率) | 年次 | 同上 |
| log y | OECD QNA / IMF IFS(実質GDP per capita) | 四半期 | ★ヴィンテージ問題(§3節) |
| logit g | SWIID / World Bank Gini / WID | 不定期(1〜5年) | SWIID は補間済み(相関構造に注意)⚖ |
| log T 代理 | WIPO 特許出願 / PWT の TFP | 年次 | proxy 選定 ⚖ |
| logit phi | ITU(インターネット利用率)ほか普及系列 | 年次 | 「どの技術の普及か」の定義 ⚖ |
| log v | ITU(モバイル/ブロードバンド契約)等 | 年次(月次は困難) | §9.1 の月次から頻度低下 → R 再設計 |
| logit tau | World Values Survey / 政府信頼サーベイ | 3〜5年 | 国により欠測大 |
| logit p | V-Dem 政治分極指数 | 年次 | §9.1 の2年毎より高頻度で有利 |
| イベント N | GDELT または ACLED | 日次→週次バッチ | ⚖ §4節 |

設計含意: v の月次観測が失われるため、E1 の観測密度より疎になる。
合成データで「年次 v」版の E1 感度確認(E1-sparse)を先に走らせ、
基準達成可能性を事前に見積もる(§7節 M7 に含む)。

## 3. GDP ヴィンテージ(§14-2)

- **Phase 2.0(提案)**: 最新確定系列による「疑似リアルタイム」ヒンドキャストで開始。
  改定の影響は「観測ノイズ sd の増分」として R に上乗せする近似。
- **Phase 2.5**: OECD Revisions Database / ALFRED 系の当時版データで
  真のリアルタイム検証へ拡張。
- ⚖ 判断: 2.0 → 2.5 の段階化で良いか(いきなり当時版に行くか)。

## 4. イベントデータとマーク対応(§14-3。本フェーズの主戦場)

- ソース比較: **ACLED**(人手コーディング、死者数あり、途上国中心、2010年代以降が厚い)
  vs **GDELT**(1979〜、全世界、機械抽出でノイズ・報道バイアス大、Goldstein scale)。
- 提案: 検証地域を ACLED カバレッジに合わせ、ACLED を一次ソース、
  GDELT を感度分析用の代替とする。⚖
- マーク ρ(残留率)との対応案: イベント規模 proxy(死者数 log スケール or
  Goldstein)を分位変換し、Beta(a_ρ, b_ρ) の条件付け(規模大 → ρ 小)に写像。
  対応関数は較正期間で推定し、検証期間には凍結。
- 「どの規模以上を強制ジャンプにするか」の閾値も較正対象(小規模イベントは
  週次カウント=ポアソン重み側で吸収)。

## 5. 対象国の提案 ⚖

- **安定国**: 日本(パラメータ仮値の出自、データ品質最高)。
- **変動国候補**(De > 1 相当が期待され、データが揃う): タイ(2006/2014 クーデター)、
  トルコ(2016)、エジプト(2011)、アルゼンチン(経済危機反復)。
  第一候補: **タイ**(イベントが明確・複数回、経済統計が比較的整備)。
- 最小構成: 安定1 + 変動1 で M8、その後 M9 で変動2〜3カ国に拡張。

## 6. 手法の拡張(第一段階からの持ち越し)

1. **固定ラグ平滑化(EnKS)**(§14-5): ジニ等の遅延・低頻度観測の活用と、
   k・tauA の事後推定改善(#0018 の残課題)を狙う。解析時に過去 L 年の
   状態も同時更新する固定ラグ版を assimilation.jl に追加。
2. **L2 較正パイプライン**(§8.2): 事前分布(LogNormal)からのアンサンブル
   ヒンドキャスト尤度による較正(ABC 風 or 反復 EnKF)。較正/検証期間の分割は事前凍結。
3. theta_p の状態拡大(#0010 で保留)は変動国ヒンドキャストで再評価。
